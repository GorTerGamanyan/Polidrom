# Polidrom
Write code IsPolidrom with extension.
Example:
```C#
using System;
using PolidromLibrary;

namespace Polidrom
{
    class Program
    {
        static void Main(string[] args)
        {
            string st = "Пал, а норов худ и дух ворона лап.";
            Console.WriteLine(st.IsPolidrom());
        }
    }
}
```
